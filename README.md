# C#图像相似度对比示例

本仓库提供了一个C#图像相似度对比的示例代码，通过结合网上的例子和灰度直方图方法计算相似度的算法，实现了图像相似度的对比功能。经过测试，该算法在图像相似度对比中达到了95%的准确率。

## 功能描述

- **图像相似度计算**：使用灰度直方图方法计算两张图像的相似度。
- **高准确率**：经过测试，该算法在图像相似度对比中表现出色，准确率高达95%。
